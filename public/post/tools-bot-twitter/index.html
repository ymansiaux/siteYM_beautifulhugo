Bot twitter sur l'ouverture du pont Chaban Delmas à Bordeaux

 Posté le 19/07/2022  |  1 minutes  |  153 mots  |  Yohann Mansiaux

Mini projet “jouet” pour tester l’implémentation d’un bot twitter.

Tous les matins, ce programme va publier l’état d’ouverture du pont Chaban Delmas à Bordeaux et les fermetures éventuelles dans la journée.

Les données d’ouverture / fermeture prévisionnelles proviennent du portail open data de Bordeaux Métropole.

La publication des tweets est réalisée via le package {rtweet}.

Pour réaliser ce programme j’ai suivi le tuto super complet du créateur du bot londonmapbot.

Pour utiliser {rtweet}, il est nécessaire d’avoir un compte développeur sur twitter de niveau Elevated. Tout est bien expliqué dans le tuto, et les liens vers la doc Twitter y sont présents.

Les ressources suivantes m’ont également été utiles pour tout bien configurer :

  • Documentation de rtweet

  • Article de R bloggers

Il s’agissait d’un premier test assez simple de l’utilisation de {rtweet}, avant d’envisager d’autres applications plus tard, peut-être plus utiles ;-)
