Package R shinyYM

Extensions pour shiny

 Posté le 01/07/2021  |  1 minutes  |  88 mots  |  Yohann Mansiaux

Ce package propose l’ajout d’extensions pour le package de développement Web shiny.

Ce package permet notamment l’utilisation d’un certain nombre de notifications proposées dans la librairie Javascript notie.

L’application de consultation des résultats des élections propose l’utilisation de ce package, pour l’utilisation de la librairie notie mais également pour l’affichage d’un écran de chargement avec le logo du datalab Bordeaux Métropole. N’hésitez pas à consulter cette appli, ainsi que son code source pour voir comment utiliser ces fonctions.

Le package R est librement téléchargeable sur ma page github.
