Ce package R propose d’appeler les webservices Bordeaux Métropole features et aggregate.

Le webservice features propose de renvoie les données d’une couche de données, en temps réel ou dans le passé, tandis que le webservice aggregate propose des calculs statistiques sur les données historisées.

Ce package est disponible en téléchargement sur le portail open data de Bordeaux Métropole.

Les sources sont disponibles ici.

Ce package est un de mes outils indispensables dans mon quotidien. La plupart des analyses que je réalise, soit sur des données internes soit sur l’open data BM, dépendent de l’utilisation de cet outil. C’est le premier package “propre” que j’ai développé et j’en suis assez content :)
