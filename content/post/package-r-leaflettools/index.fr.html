Ce package propose l’ajout d’extensions pour le package de cartographie leaflet.

En l’état, une seule extension est implémentée. La fonction addBezierCurve permet d’ajouter des courbes incurvées entre 2 points.

Le code est largement inspiré d’un code JavaScript.

Le package R est librement téléchargeable sur ma page github.
