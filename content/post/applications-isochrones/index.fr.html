Cette application propose d’utiiser le webservice de géocodage de Bordeaux Métropole, de calculer des isochrones autour de cette adresse géolocalisée (zone présente à x minutes à pied / à vélo / en voiture), et d’afficher les équipements publics présents dans la zone calculée.

Cette application a été intéressante à réaliser pour plusieurs raisons :

  • L’utilisation du package R xml2 pour parser les résultats du webservice de géocodage de Bordeaux Métropole

  • L’utilisation du package R osrm pour le calcul des isochrones

  • La manipulation de données géographiques pour calculer les équipements publics présents dans l’isochrone calculé

  • L’utilisation combiné des grids CSS et des flexbox pour le design de l’application.
